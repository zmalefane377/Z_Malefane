Kerf Testing

  1. Design a kerf set up according to the size of the wood you will use...in this case we used Coreldraw for the Design
  2. If you are unsure of the size you can measure it
  3. Make sure you have different hairlines or line measurements so that you can see which one is better to use on the material you will use
    1. The hairline
    2. The 0.5pt
  4. Cut out the pieces and note down the analysis for later reference
  5. Fit in the Wood piece to see where it best fits
